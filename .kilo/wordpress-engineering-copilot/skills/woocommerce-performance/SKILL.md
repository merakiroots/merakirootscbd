# WooCommerce Performance

Use this skill when writing or reviewing WooCommerce PHP code that may benefit from cache priming.

## Focus Areas

- Post cache priming for collections of post-based objects such as products and orders
- Cache priming when rendering product lists with images
- Option cache priming when reading multiple non-autoloaded options in a loop or method
