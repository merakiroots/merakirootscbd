# WooCommerce Email Editor Development

Use this skill when setting up local development for the email editor, working on email templates or transactional emails, modifying the email editor PHP or JS packages, or testing email sending functionality.

## Key Guidance

- Start the WooCommerce `wp-env` environment before working on the editor
- Use the admin watcher when changing email editor JS packages
- Enable the block email editor feature in WooCommerce settings
- Use Mailpit as a local SMTP server for email testing
- Rebuild the email editor package and run its PHP tests when needed
