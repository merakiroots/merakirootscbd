# WooCommerce Development Cycle

This skill provides guidance for the WooCommerce development workflow, including running tests, code quality checks, and troubleshooting.

## Instructions

Follow these guidelines for WooCommerce development workflow:

1. Run relevant tests after making code changes
2. Run linting and type checking for changed code
3. Fix linting issues before committing
4. Use project-specific PHP and JS i18n patterns
5. Lint markdown when markdown files change

## Development Workflow

1. Make code changes
2. Run relevant tests
3. Run linting and type checking
4. Fix any issues
5. Commit changes only after tests pass
