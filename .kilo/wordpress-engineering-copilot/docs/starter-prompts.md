# Starter Prompts

1. Review a theme build
   - description: Audit theme structure and UX.
   - prompt: Review this WordPress theme approach for frontend quality, maintainability, accessibility, and block-theme compatibility. Flag issues and suggest the best implementation path.

2. Plan data architecture
   - description: Model backend and database choices.
   - prompt: Help me design the architecture for a WordPress feature, including whether to use core data structures or custom tables, what hooks and APIs to use, and how to keep it maintainable.

3. Design WooCommerce flow
   - description: Plan a store integration.
   - prompt: Help me design and implement a WooCommerce customization or extension, including checkout, cart, order handling, HPOS compatibility, and performance considerations.

4. Run a security review
   - description: Audit WordPress or WooCommerce code.
   - prompt: Review this WordPress or WooCommerce code for security risks. Prioritize real vulnerabilities, explain impact, and recommend the safest fixes.
