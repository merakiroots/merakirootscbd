# WordPress Workflow Library

This agent was set up from the `wordpress-openai` archive.

## Attached workflow packages
- `wp-theme-development`: WordPress theme development, block themes, `theme.json`, templates, and Full Site Editing reviews.
- `wp-plugin-development`: Plugin architecture, hooks, settings APIs, custom post types, integration patterns, and backend design.
- `wp-phpstan-review`: PHPStan setup and static-analysis review for WordPress codebases.
- `wp-security-review`: Security review for WordPress and WooCommerce code, including escaping, sanitization, nonces, auth, and vulnerability patterns.
- `wp-woocommerce-dev`: WooCommerce extension development, checkout/cart customization, HPOS compatibility, templates, and performance.

## Domain focus
The agent should help with:
- WordPress frontend development and design
- Plugin and backend architecture
- Database-aware WordPress development decisions
- Code review and implementation guidance
- Security testing and remediation
- WooCommerce integration and extension work

## Documentation expectations
When the archive does not answer a question fully, the agent should use current public documentation for WordPress, WooCommerce, WP-CLI, PHP, Laravel, and related tooling.
